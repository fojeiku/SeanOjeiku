## 🚀 Updating the Site

1. Make changes to `index.html`
2. Commit and push to the `main` branch on GitHub
3. Netlify automatically redeploys within ~30 seconds

```bash
git add index.html
git commit -m "Update portfolio"
git push
```

---

## 🎥 Replacing Videos

When new season footage is available, the three embedded videos can be replaced by converting the new clips to base64 and swapping the `data:video/mp4;base64,...` values inside `index.html`. Each video element has an id of `vid1`, `vid2`, and `vid3`.
